## 🔍 核心概念速览

### 什么是 Mizarα？

Mizarα 是一个创新的量化交易框架，采用**"检索即预测"**的范式：

```
传统ML: 历史数据 → 训练模型 → 预测未来
Mizarα:  历史数据 → 构建向量库 → 检索相似 → 统计预测
```

### 核心优势

✅ **非参数化方法** — 不假设数据分布，避免模型偏差  
✅ **强可解释性** — 每个预测都有历史案例支撑  
✅ **自适应更新** — 新数据自动融入，无需重新训练  
✅ **实时预测** — 单次查询响应时间约200ms  

### 工作原理

```
当前市场状态 → 特征向量化 → 检索历史相似 → 统计后续走势 → 生成预测
```

## 🛠️ 快速使用

### 安装

```bash
pip install mizar-alpha
```

### 预测

```bash
# 命令行
mizar predict --stock 000001 --top-k 10
```

```python
# Python
from mizar import PredictionService

service = PredictionService()
result = service.predict('000001', '2026-04-21')
print(f"上涨概率: {result.up_probability:.2%}")
```

### 回测

```bash
mizar backtest --config config/backtest.yaml
```
