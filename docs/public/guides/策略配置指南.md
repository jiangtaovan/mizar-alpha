### 2. 归一化配置

```yaml
normalization:
  method: minmax  # 或 zscore
  
  # minmax: 缩放到[0,1]
  # 适用场景: 特征范围差异大
  
  # zscore: 标准化为均值0、方差1
  # 适用场景: 特征近似正态分布
```

### 3. PCA降维配置

```yaml
pca:
  n_components: 0.95  # 保留95%方差
  
  # 可选值:
  # - 0.90-0.95: 推荐，平衡性能和精度
  # - 0.95-0.99: 高精度，维度较多
  # - 0.80-0.90: 高性能，损失部分信息
```

## 🎯 相似度计算策略

### 距离度量方法

```yaml
similarity:
  metric: euclidean  # 距离度量
  
  # 可选方法:
  # - euclidean: 欧氏距离（默认）
  # - cosine: 余弦相似度
  # - manhattan: 曼哈顿距离
  # - minkowski: 闵可夫斯基距离
```

### 加权策略

```yaml
weighting:
  method: distance  # 加权方法
  
  # 可选方法:
  # - distance: 距离加权（推荐）
  #   权重 = 1 / (distance + epsilon)
  
  # - uniform: 均匀权重
  #   所有相似样本权重相同
  
  # - exponential: 指数加权
  #   权重 = exp(-distance * alpha)
```

### Top-K选择

```yaml
query:
  top_k: 10  # 返回相似样本数量
  
  # 建议范围:
  # - 5-10:  精确预测，样本少
  # - 10-20: 平衡选择（推荐）
  # - 20-50: 统计稳定，噪声多
```

## 📊 置信度评估

### 置信度因子

```python
confidence_score = (
    0.40 * up_probability +      # 上涨概率权重40%
    0.25 * sample_quality +      # 样本质量权重25%
    0.20 * similarity_strength + # 相似度强度20%
    0.15 * market_regime +       # 市场状态15%
)
```

### 动态权重调整

```yaml
confidence:
  # 基础权重
  base_weights:
    up_probability: 0.40
    sample_quality: 0.25
    similarity_strength: 0.20
    market_regime: 0.15
  
  # 市场状态调整
  regime_adjustments:
    bull:
      up_probability: 0.35    # 牛市降低概率权重
      market_regime: 0.25     # 提高市场状态权重
    bear:
      up_probability: 0.45    # 熊市提高概率权重
      sample_quality: 0.30    # 提高样本质量权重
```

### 置信度阈值

| 置信度 | 信号强度 | 操作建议 |
|--------|---------|---------|
| > 0.75 | 强买入 | 满仓或加仓 |
| 0.65-0.75 | 买入 | 正常建仓 |
| 0.55-0.65 | 弱买入 | 轻仓试探 |
| 0.45-0.55 | 中性 | 持有观望 |
| 0.35-0.45 | 弱卖出 | 减仓 |
| < 0.35 | 强卖出 | 清仓或做空 |

## 💼 交易策略设计

### 策略1: 高置信度选股

```python
def high_confidence_strategy(predictions, threshold=0.70):
    """选择高置信度上涨标的"""
    candidates = []
    
    for symbol, pred in predictions.items():
        if pred.confidence_score >= threshold:
            candidates.append({
                'symbol': symbol,
                'confidence': pred.confidence_score,
                'expected_return': pred.avg_return,
                'up_probability': pred.up_probability
            })
    
    # 按置信度排序
    candidates.sort(key=lambda x: x['confidence'], reverse=True)
    return candidates[:10]  # 返回前10
```

### 策略2: 多周期信号融合

```python
def multi_timeframe_signals(symbol, timeframes=['1d', '5d', '10d']):
    """多周期信号融合"""
    signals = {}
    
    for tf in timeframes:
        pred = service.predict(symbol, timeframe=tf)
        signals[tf] = pred.confidence_score
    
    # 加权平均
    weights = {'1d': 0.5, '5d': 0.3, '10d': 0.2}
    fused_signal = sum(signals[tf] * weights[tf] for tf in timeframes)
    
    return {
        'fused_signal': fused_signal,
        'individual_signals': signals,
        'consensus': all(s > 0.6 for s in signals.values())
    }
```

### 策略3: 动态仓位管理

```python
def position_management(confidence_score, portfolio_value):
    """基于置信度的仓位管理"""
    
    # Kelly公式简化版
    if confidence_score > 0.70:
        position_size = 0.20  # 20%仓位
    elif confidence_score > 0.65:
        position_size = 0.15  # 15%仓位
    elif confidence_score > 0.60:
        position_size = 0.10  # 10%仓位
    else:
        position_size = 0.05  # 5%仓位
    
    # 单笔最大仓位限制
    position_size = min(position_size, 0.25)
    
    return portfolio_value * position_size
```

backtest:

  # 风控参数
  risk_management:
    stop_loss: -0.05           # 止损5%
    take_profit: 0.10          # 止盈10%

from itertools import product

# 定义参数空间
param_grid = {
    'top_k': [5, 10, 15, 20],

    'pca_components': [0.90, 0.95, 0.99]

# 网格搜索
best_sharpe = -999
best_params = None

for params in product(*param_grid.values()):
    top_k, threshold, pca = params

    result = run_backtest(

## ⚠️ 实战注意事项

### 数据质量

- ✅ 确保技术指标计算正确
- ✅ 处理停牌、除权除息等特殊情况
- ✅ 避免未来函数（look-ahead bias）

### 过拟合风险

- ❌ 不要过度优化参数
- ✅ 使用样本外数据验证
- ✅ 参数应该在经济意义上合理

### 市场变化

- ⚠️ 历史模式不一定重复
- ⚠️ 市场结构变化时重新评估策略
- ✅ 定期更新向量数据库

### 风控优先

- 🛡️ 设置止损和最大回撤限制
- 🛡️ 单笔仓位不超过总资金20%
- 🛡️ 监控策略表现，及时止损失效策略
