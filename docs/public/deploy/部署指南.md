## 🐳 Docker 部署

### 方式一：使用 Dockerfile

#### 1. 构建镜像

```bash
# 构建镜像
docker build -t mizar-alpha:latest .

# 查看镜像
docker images | grep mizar-alpha
```

#### 2. 运行容器

```bash
# 基础运行
docker run -d \
  --name mizar-service \
  -p 8000:8000 \
  -v $(pwd)/data:/app/data \
  -v $(pwd)/models:/app/models \
  mizar-alpha:latest

# 带环境变量运行
docker run -d \
  --name mizar-service \
  -p 8000:8000 \
  -e MIZAR_LOG_LEVEL=INFO \
  -e MIZAR_DB_HOST=localhost \
  -v $(pwd)/data:/app/data \
  -v $(pwd)/models:/app/models \
  mizar-alpha:latest
```

#### 3. 查看日志

```bash
# 实时日志
docker logs -f mizar-service

# 最近100行
docker logs --tail 100 mizar-service
```

### 方式二：使用 Docker Compose

#### 1. 配置文件

```yaml
# docker-compose.yml
version: '3.8'

services:
  mizar-api:
    build: .
    container_name: mizar-api
    ports:
      - "8000:8000"
    volumes:
      - ./data:/app/data
      - ./models:/app/models
      - ./config:/app/config
    environment:
      - MIZAR_LOG_LEVEL=INFO
      - MIZAR_WORKERS=4
    restart: unless-stopped
    networks:
      - mizar-network

  # 可选：添加监控服务
  prometheus:
    image: prom/prometheus:latest
    ports:
      - "9090:9090"
    volumes:
      - ./prometheus.yml:/etc/prometheus/prometheus.yml
    networks:
      - mizar-network

  grafana:
    image: grafana/grafana:latest
    ports:
      - "3000:3000"
    environment:
      - GF_SECURITY_ADMIN_PASSWORD=admin
    networks:
      - mizar-network

networks:
  mizar-network:
    driver: bridge
```

#### 2. 启动服务

```bash
# 启动所有服务
docker-compose up -d

# 查看状态
docker-compose ps

# 查看日志
docker-compose logs -f mizar-api

# 停止服务
docker-compose down

# 停止并删除数据卷
docker-compose down -v
```

## ☁️ 生产环境部署

### 1. 服务器要求

| 组件 | 最低配置 | 推荐配置 |
|------|---------|---------|
| CPU | 4核 | 8核+ |
| 内存 | 8GB | 16GB+ |
| 磁盘 | 100GB SSD | 500GB NVMe |
| 网络 | 100Mbps | 1Gbps |

### 2. 系统优化

```bash
# Linux 系统参数优化
sudo sysctl -w net.core.somaxconn=65535
sudo sysctl -w net.ipv4.tcp_max_syn_backlog=65535
sudo sysctl -w vm.swappiness=10

# 文件描述符限制
echo "* soft nofile 65535" | sudo tee -a /etc/security/limits.conf
echo "* hard nofile 65535" | sudo tee -a /etc/security/limits.conf
```

### 3. Nginx 反向代理

```nginx
# /etc/nginx/sites-available/mizar
upstream mizar_backend {
    server 127.0.0.1:8000;
}

server {
    listen 80;
    server_name api.mizar-alpha.com;

    # 重定向到 HTTPS
    return 301 https://$server_name$request_uri;
}

server {
    listen 443 ssl http2;
    server_name api.mizar-alpha.com;

    ssl_certificate /etc/ssl/certs/mizar.crt;
    ssl_certificate_key /etc/ssl/private/mizar.key;

    # 安全头
    add_header X-Frame-Options DENY;
    add_header X-Content-Type-Options nosniff;
    add_header Strict-Transport-Security "max-age=31536000";

    location / {
        proxy_pass http://mizar_backend;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;

        # 超时设置
        proxy_connect_timeout 60s;
        proxy_send_timeout 60s;
        proxy_read_timeout 60s;
    }

    # 静态文件
    location /static {
        alias /app/static;
        expires 30d;
    }

    # 日志
    access_log /var/log/nginx/mizar_access.log;
    error_log /var/log/nginx/mizar_error.log;
}
```

### 4. Systemd 服务

```ini
# /etc/systemd/system/mizar.service
[Unit]
Description=Mizarα Prediction Service
After=network.target

[Service]
Type=simple
User=mizar
Group=mizar
WorkingDirectory=/opt/mizar
Environment=PATH=/opt/mizar/venv/bin
ExecStart=/opt/mizar/venv/bin/mizar serve --host 127.0.0.1 --port 8000
Restart=always
RestartSec=5

# 资源限制
LimitNOFILE=65535
MemoryMax=4G

[Install]
WantedBy=multi-user.target
```

```bash
# 启用服务
sudo systemctl daemon-reload
sudo systemctl enable mizar
sudo systemctl start mizar

# 查看状态
sudo systemctl status mizar

# 查看日志
sudo journalctl -u mizar -f
```

## 🔒 安全配置

### 1. API 认证

```python
# 添加 API Key 验证
from fastapi import Security, HTTPException
from fastapi.security import APIKeyHeader

api_key_header = APIKeyHeader(name="X-API-Key")

async def verify_api_key(api_key: str = Security(api_key_header)):
    if api_key not in VALID_API_KEYS:
        raise HTTPException(status_code=403, detail="Invalid API Key")
```

### 2. 速率限制

```python
from slowapi import Limiter
from slowapi.util import get_remote_address

limiter = Limiter(key_func=get_remote_address)

@app.post("/predict")
@limiter.limit("100/minute")
async def predict(request: Request):
    pass
```

### 3. 数据备份

```bash
#!/bin/bash
# backup.sh - 每日备份脚本

DATE=$(date +%Y%m%d)
BACKUP_DIR="/backup/mizar"

# 备份向量数据库
tar -czf ${BACKUP_DIR}/chroma_db_${DATE}.tar.gz data/chroma_db/

# 备份模型文件
tar -czf ${BACKUP_DIR}/models_${DATE}.tar.gz models/

# 备份配置文件
tar -czf ${BACKUP_DIR}/config_${DATE}.tar.gz config/

# 保留30天备份
find ${BACKUP_DIR} -name "*.tar.gz" -mtime +30 -delete
```

```bash
# 添加到 crontab
0 2 * * * /opt/mizar/scripts/backup.sh
```
