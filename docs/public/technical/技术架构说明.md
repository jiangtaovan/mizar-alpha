# Mizarα 技术架构说明

> 深入了解向量相似性检索预测系统的核心设计

---

## 📋 系统概述

Mizarα 采用**"检索即预测"**的创新范式，区别于传统的机器学习预测方法。

### 核心理念

```
传统ML: 历史数据 → 训练模型 → 预测未来
Mizarα:  历史数据 → 构建向量库 → 检索相似 → 统计预测
```

### 设计哲学

1. **模式重复假设**：相似的市场状态倾向于产生相似的后续走势
2. **非参数化方法**：不假设数据分布，避免模型偏差
3. **可解释性强**：每个预测都有历史案例支撑
4. **自适应更新**：新数据自动融入，无需重新训练

## 🔬 核心算法

### 1. 特征工程流程

```
原始数据 → 特征提取 → 归一化 → PCA降维 → 特征向量
```

#### 步骤详解

**Step 1: 特征提取**

```python
# 从原始K线数据计算技术指标
features = {
    'MA_SMA_5': calculate_sma(close, window=5),
    'RSI_12': calculate_rsi(close, window=12),
    'MACD': calculate_macd(close),
    # ... 30-50个特征
}
```

**Step 2: 归一化**

```python
# Min-Max归一化
X_normalized = (X - X_min) / (X_max - X_min)

# 或Z-Score标准化
X_standardized = (X - mean) / std
```

**重要性**：消除不同特征量纲差异，避免大数值特征主导距离计算。

**Step 3: PCA降维**

```python
from sklearn.decomposition import PCA

# 保留95%方差
pca = PCA(n_components=0.95)
X_reduced = pca.fit_transform(X_normalized)

# 通常从50维降至15-25维
```

**优势**：
- 去除特征间冗余
- 加速相似度检索
- 降低噪声影响

### 2. 向量存储与索引

#### ChromaDB架构

```python
import chromadb

# 初始化向量数据库
client = chromadb.PersistentClient(path='data/chroma_db')

collection = client.create_collection(
    name='market_states',
    metadata={'hnsw:space': 'l2'}  # L2距离（欧氏距离）
)

# 添加向量
collection.add(
    embeddings=vectors,        # 特征向量
    documents=metadata,        # 元数据（日期、股票等）
    ids=unique_ids             # 唯一标识
)
```

#### HNSW索引

Mizarα 使用 **HNSW** (Hierarchical Navigable Small World) 索引：

- **查询复杂度**: O(log N)
- **构建复杂度**: O(N log N)
- **适用场景**: 高维向量近似最近邻搜索

**参数配置**：
```yaml
hnsw:
  M: 16              # 最大连接数
  ef_construction: 200  # 构建时搜索深度
  ef_search: 50      # 查询时搜索深度
```

### 3. 相似度检索算法

#### 距离度量

```python
# 欧氏距离（默认）
def euclidean_distance(x, y):
    return np.sqrt(np.sum((x - y) ** 2))

# 余弦相似度
def cosine_similarity(x, y):
    return np.dot(x, y) / (np.linalg.norm(x) * np.linalg.norm(y))
```

#### 加权统计

```python
def weighted_prediction(similar_states, weights='distance'):
    """基于相似状态的加权预测"""
    
    if weights == 'distance':
        # 距离加权：距离越近权重越大
        w = 1 / (distances + 1e-8)
    elif weights == 'uniform':
        # 均匀权重
        w = np.ones(len(distances))
    elif weights == 'exponential':
        # 指数加权
        w = np.exp(-distances * alpha)
    
    # 加权平均收益
    avg_return = np.average(returns, weights=w)
    
    # 上涨概率
    up_probability = np.sum(w[returns > 0]) / np.sum(w)
    
    return {
        'avg_return': avg_return,
        'up_probability': up_probability,
        'sample_size': len(similar_states)
    }
```

### 4. 置信度评估

```python
def calculate_confidence(prediction, market_context):
    """多维度置信度评估"""
    
    # 1. 上涨概率（40%权重）
    score_prob = prediction['up_probability']
    
    # 2. 样本质量（25%权重）
    #    - 样本数量充足度
    #    - 相似度强度
    sample_quality = min(prediction['sample_size'] / 20, 1.0)
    
    # 3. 相似度强度（20%权重）
    #    - Top-1距离
    #    - 平均距离
    similarity_strength = 1 / (1 + prediction['avg_distance'])
    
    # 4. 市场状态（15%权重）
    #    - 趋势一致性
    #    - 波动率匹配
    market_regime = assess_regime_match(market_context)
    
    # 综合置信度
    confidence = (
        0.40 * score_prob +
        0.25 * sample_quality +
        0.20 * similarity_strength +
        0.15 * market_regime
    )
    
    return confidence
```

## 🔧 性能优化

### 1. 查询优化

```python
# 批量查询优于循环
def batch_predict(symbols, date):
    """批量预测优化"""
    
    # 1. 批量计算特征
    features = compute_features_batch(symbols, date)
    
    # 2. 批量归一化
    normalized = scaler.transform(features)
    
    # 3. 批量PCA
    reduced = pca.transform(normalized)
    
    # 4. 批量查询（ChromaDB支持）
    results = collection.query(
        query_embeddings=reduced.tolist(),
        n_results=10
    )
    
    return results
```

### 2. 缓存策略

### 3. 增量更新

```python
def incremental_update(new_data):
    """每日增量更新"""
    
    # 1. 计算新数据特征
    features = compute_features(new_data)
    
    # 2. 使用已有Scaler和PCA
    normalized = scaler.transform(features)
    reduced = pca.transform(normalized)
    
    # 3. 添加到ChromaDB
    collection.add(
        embeddings=reduced,
        documents=new_data,
        ids=new_ids
    )
```
